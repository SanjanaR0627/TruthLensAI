# 📰 TruthLens AI
### AI-Powered Fake News Detection System

TruthLens AI is an end-to-end Machine Learning web application that detects whether a news article is **Real** or **Fake** using Natural Language Processing (NLP) and Machine Learning techniques.

The application analyzes the text entered by the user, predicts its authenticity, and provides additional insights such as confidence score, trust score, risk level, sentiment analysis, and text statistics through an interactive Streamlit interface.

# 📌 Problem Statement

With the rapid growth of digital media and social networking platforms, misinformation spreads faster than ever before. It has become increasingly difficult for users to distinguish between authentic and misleading news.

TruthLens AI addresses this challenge by providing an intelligent system capable of analyzing news content and estimating its credibility using Machine Learning.

---

# ✨ Features

### 📰 Fake News Detection
Predicts whether a news article is **Real** or **Fake** using a trained Machine Learning model.

### 📊 Confidence Score
Displays the confidence level of the model's prediction.

### 🛡 Trust Score
Provides a reliability score to help users understand the credibility of the news article.

### ⚠ Risk Analysis
Classifies articles into different risk categories based on prediction confidence.

### 😊 Sentiment Analysis
Determines whether the news content has Positive, Negative, or Neutral sentiment.

### 📈 Prediction Probability
Shows prediction probabilities to make model outputs more interpretable.

### 📄 Text Statistics
Displays:
- Word Count
- Character Count
- Sentence Count

### 📂 Prediction History
Maintains a history of previously analyzed articles during the session.

### 📥 Download Report
Allows users to download their prediction history in CSV format.

### 🌐 Interactive Web Interface
Built using Streamlit for a clean, responsive, and user-friendly experience.

# 🚀 How It Works

1. User enters a news article.
2. Text is cleaned and preprocessed.
3. The TF-IDF Vectorizer converts text into numerical features.
4. The trained Machine Learning model analyzes the article.
5. The application predicts whether the news is **Real** or **Fake**.
6. Additional analytics such as confidence, trust score, risk level, and text statistics are displayed.
