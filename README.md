AI-POWERED Explainable Trust Assessment for Social Media Content
# 🔍 TruthLens AI

TruthLens AI is an NLP-based Machine Learning project that analyzes social media news and predicts whether it is Fake or Real.

Unlike traditional fake news detectors, TruthLens AI also generates a Trust Score and explains why the prediction was made.

## Features

- Fake News Detection
- NLP Preprocessing
- TF-IDF Vectorization
- Logistic Regression
- Naive Bayes
- Random Forest
- Linear SVM
- Trust Score Engine
- Explainable AI

## Technologies

- Scikit-Learn
- NLTK
- Pandas
- NumPy
- Matplotlib
- Streamlit
